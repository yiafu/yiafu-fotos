Fotografía

Todos estos paisajes son muy bonitos para hacerse fotos.Podeis elegir otros sitios que no sean estos para la sesión de fotos. Hos explicare en cada foto por que es bonito ir y tener unas fotos super bonitas.Hago todo tipo de fotografía para bodas, comuniones, recien nacidos... Lo que cuesta depende de donde es el sitio.

Esta montaña tiene unas vista desde arriba impresionante y además con las apuestas de sol es como si estuvieses casi al lado del sol.

Este es el Mar Mayor, he elegido este sitio porque al tener un poco de brisa a las chicas si se llevan un vestido largo suelto hace una efecto como si estuvieran solo paseando y de repente sin que se entere le hacen una foto.

Este es el puerto,me gusta este sitio porque tiene el significado de la ballena
